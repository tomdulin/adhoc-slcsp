# Adhoc::Slcsp
Short description and motivation.

## setup
You may need to provide permission to the executa
```ruby
sudo chmod 755 'bin/slcsp'
```

## usage
Simply run
```ruby
bin/slcsp
```

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'adhoc-slcsp'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install adhoc-slcsp
```
